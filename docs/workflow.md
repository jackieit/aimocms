# 工作流投稿说明 

  * 涉及数据库表  
  
    ``` workflow        工作流表 ``` 
    ``` workflow_step   工作流步骤表 ``` 
    ``` workflow_state  工作流操作状态表  ``` 
    ``` workflow_log    工作流操作日志  ``` 
     
  
  * 工作流操作流程  
    1. 创建工作流状态码(部分已经内置)  
    2. 创建工作流  
    3. 创建工作流步骤,尤其要定义之前与之后操作的状态代码  
    4. 目前工作流步骤无法调整,可通过编辑修改来实现顺序调整  
    5. 根据工作流步骤中的用户角色与状态代码来决定对当前用户是否显示及相应可进行操作  
    6. 操作时进行相应的操作日志
